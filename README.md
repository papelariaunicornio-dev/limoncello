# Limoncello — Friday Night

Calculadora de produção de limoncello. Página única estática (vanilla JS, offline-first, localStorage).

- Modo "Receita rápida": receita de mistura a partir da infusão medida (ou do volume final desejado).
- Modo "Completa": todas as fórmulas da planilha Friday Night, por grupo colapsável, incl. calculadora de Spritz.

Deploy: Coolify (nginx estático) em https://limoncello.mannato.com
